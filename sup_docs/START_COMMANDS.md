# Start Commands

Windows PowerShell friendly commands for this repository:

## Install

```powershell
npm.cmd install
```

## Run Development Server

```powershell
npm.cmd run dev
```

## Build The Extension

```powershell
npm.cmd run build
```

## Typecheck

```powershell
npm.cmd run typecheck
```

## Lint

```powershell
npm.cmd run lint
```

## Suggested Local Flow

```powershell
npm.cmd install
npm.cmd run typecheck
npm.cmd run lint
npm.cmd run build
```
